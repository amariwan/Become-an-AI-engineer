# 02 vorgefertigte modelle – Block 3

## Beschreibung

MoE-Modelle bestehen aus vielen spezialisierten Sub-Netzwerken (Experts)
und einem Router, der für jeden Token nur einen Bruchteil der Experten
aktiviert. Dadurch erreichen sie die Kapazität großer Modelle bei den
Kosten kleinerer.

## Bildbeschreibung

Multimodale Architektur: Drei Input-Pfade von links – Text (blau), Image (grün), Audio (orange). Jeder führt in seinen Encoder (Tokenizer, Vision Encoder, Audio Encoder). Alle Encoder münden in einen zentralen Fusion Encoder (violett). Vom Fusion Encoder Pfeil zum Decoder/LLM (rechts, grosses Kästchen). Moderner Datenfluss von links nach rechts, farblich kodierte Modalitäten. Für Buch geeignet: sauber, erklärend, hierarchisch.
