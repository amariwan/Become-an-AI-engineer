# 02 vorgefertigte modelle – Block 2

## Beschreibung

Das Resultat sind Modelle wie GPT-4o, Claude, Gemini oder Llama -- sie alle wurden auf diese Weise vortrainiert. Die eigentliche Kunst liegt dann im \textbf{Fine-Tuning} und der \textbf{Integration} dieser Foundation Models in Anwendungslogik.

## Bildbeschreibung

Zwei-Spalten-Vergleich: Links 'Ohne RAG' – User-Frage → LLM → Antwort (nur aus Training). Rechts 'Mit RAG' – User-Frage → paralleler Pfad: FAQ/DB → Embedding → Vector Search → Relevante Chunks → LLM mit Kontext → fundierte Antwort. Der rechte Pfad ist farblich hervorgehoben. Pfeile verbinden die Schritte. Layout: links einfach, rechts detailliert.
